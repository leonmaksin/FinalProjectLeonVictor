Leon and Victor's CS Final Project

About

This is a database containing the top 100 songs on the billboard 100 from 1965 to 2015

This project give you the results of the songs that you search by in a table including the rank, song name, artist name, year, number of lyrics, and number of unique lyrics

You can also check off songs to form a graph

Instructions

The homepage allows the user to choose from five different categories:

  1. All Fields - This will search in all the other four fields
  2. Song Name - This will search by the name songs
  3. Artist - This will search by the artist of songs
  4. Year - This will search by the year the songs was on the billboard 100
  5. Rank - This will search by the rank of the songs on the billboard 100
More info on help page

After they choosing a category, the user should then type their input (what they will search by)

The webpage will present a list of songs with statistics, this is where the main data is shown

You can then check off songs to graph (or check all) and graph them in comparison on six different graphs

Working Features

A list of all of the features:

  • Home Page:
    • A search bar allowing you to search by certain field
    • An advanced search algorithm that isn't case sensitive and auto-completes
  • Help Page:
    • A help page that gives directions on how to use the site
  • Table Page:
    • Loads song data into a table
    • Each row is a different song
    • The columns are:
      1. Rank
      2. Song Name
      3. Artist
      4. Year
      5. Number of lyrics
      6. Number of unique lyrics
      7. A checkbox that allows you to choose which song to graph
  • Graph Page:
    • Six sperate graphs that display a comparison of different song statistics using a scatter plot
    • Statistics include:
      • Rank
      • Year
      • Number of Lyrics
      • Number of Unique Lyrics
  • Other Miscelaneous
